# Project Roadmap

This roadmap outlines the planned evolution of the authentication module and future backend components.

---

## 🔹 v0.2.0 — Authorization Module
- Role-based authorization
- Permission model
- Policy-based access control
- Integration with authentication module

## 🔹 v0.3.0 — Users Module
- User profile management
- User CRUD operations
- Domain events for user lifecycle

## 🔹 v0.4.0 — Email Verification
- Email verification tokens
- Verification flow
- Resend verification email

## 🔹 v0.5.0 — Password Reset
- Password reset tokens
- Reset flow
- Token expiration and security rules

## 🔹 v0.6.0 — Notifications Module
- Email notifications
- Template system
- Integration with SendGrid or SMTP

## 🔹 v0.7.0 — Audit Logging
- Login attempts logging
- Token usage logging
- Security event tracking

---

This roadmap may evolve as the architecture grows.
